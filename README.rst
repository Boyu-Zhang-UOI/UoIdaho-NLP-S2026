Natural Language Processing — University of Idaho (Spring 2026)
================================================================

Course materials for **CS 4/574: Natural Language Processing** at the University of Idaho.

📖 **View the course site:** https://Boyu-Zhang-UOI.github.io/UoIdaho-NLP-S2026/

Course Sections
---------------

1. `Introduction to NLP <https://Boyu-Zhang-UOI.github.io/UoIdaho-NLP-S2026/notebooks/01-introduction>`_ — What NLP is, core tasks, and your first pipeline.
2. `Text Preprocessing <https://Boyu-Zhang-UOI.github.io/UoIdaho-NLP-S2026/notebooks/02-text-preprocessing>`_ — Tokenization, stemming, lemmatization, stopwords, regex.
3. `Text Representation <https://Boyu-Zhang-UOI.github.io/UoIdaho-NLP-S2026/notebooks/03-text-representation>`_ — Bag-of-Words, TF-IDF, Word2Vec embeddings.
4. `Text Classification <https://Boyu-Zhang-UOI.github.io/UoIdaho-NLP-S2026/notebooks/04-text-classification>`_ — Naïve Bayes, Logistic Regression, evaluation metrics.
5. `Language Models & Transformers <https://Boyu-Zhang-UOI.github.io/UoIdaho-NLP-S2026/notebooks/05-language-models-transformers>`_ — N-grams, attention, Transformers, BERT, GPT.

Each section is presented in a **Jupyter Notebook style** with explanations, Python code, and expected output.

Local Development
-----------------

    cd docs
    bundle install
    bundle exec jekyll serve
